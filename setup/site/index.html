上記の内容のように、ISOファイルまたはDVDファイル等をダウンロードし、上記に記載した仮想マシン（vmbox）の中へとOSをインストールしていきます。

アップデートをかけます

sudo apt-get update

アップグレードをかけます

sudo apt-get upgrade

ファイヤーウォールを切ります

sudo ufw disable

SELINUXを切ります。

sudo service apparmor stop

sudo apt install net-tools

sudo apt install git

だいたいこれで、一通りは不自由ないかと思われます。

lamp

lamp2

Linux、Apache、MySQL、PHPの環境でWebシステムを開発すること。それぞれの頭文字をとってLAMPと言います。開発が主流となっている開発環境です。二年前に初めて導入したのですが、実際難しくは無かったです。ＷindowsではxamppをmacosではMAMPをインストールすることで完了します。本サイトではlinuxos(ubuntu)での構築方法を記載いたしております。

Laravel

Laravel2

PHPのフルスタックフレームワークで最近流行っております。特徴について➡⓵設定をあとから色々と追加しなくてはならないため、初期設定の難しさがある。➁Eloquent ORM(アクティブレコードによるデーター操作の実装)の書き方に迷う。⓷ディレクトリ構成が自由なので柔軟性がある。⓸反面、設定や書き方等がばらつきやすく、最初にさわるフレームワークとしてはあまりよくないかもしれません。

django

django2

　　　　　　　　

　　　　　　　

pythonでweb開発をするためのフレームワークです。もともとpythonは機械学習（ai）やディープラーニングで用いられるプログラミング言語。動作スピードが早いということで、web開発のバックエンドに近年では使用されるようになったそうです。環境構築にAnacondaを使用すると楽です。導入する際使用するディストリビューションのAnaconda(アナコンダ)は、「データサイエンス」や「機械学習」の用途でも使用します。インストールは楽ですが、時間がかかるのでご注意。

ruby on rails

ruby on rails2

　

Ruby on railsとはRubyのフルスタックフレームワークです。こちらは世界というより日本で流行っております。その理由は恐らくRuby自体を開発したのが日本人、にもかかわらず…Windowsとの相性が良くない。環境構築の際、一度もうまくいかなかったのです（笑）ひとことで言うと魔法のようなフレームワークです。リレーションシップやActive Record等の機能は他の追随を許さないといっても過言ではないでしょう。Active Record=企業のシステム開発において頻繁に認められるデザインパターン。(crud)クラウドともいう。c=crete=新規作成、r=read=読込、u=update=更新。d=delete=削除。※編集＝editはupdateに含まれます。

Vue.js

Vue.js2

私もよく使っているjqueryというライブラリはDOM操作はしやすいものの、大規模化するとコードの修正や可読に疲れてしまうことがあります。 大規模化しても記述をわかりやすくするためテンプレートを使用する。そこで注目されたのが、Vue.jsです。VueのTemplatesのメリットは、ほぼHTML・CSSの普通の書き方と同じように書くことができること。【DOM＝（Document Object Model）とは、xmlやhtmlの各要素、pタグやimgタグの類の要素にアクセスする仕組みのことです。】またRailsやLaravel等のフレームワークに組み込むこともできます。記述も短くなるのでより短時間でハイレベルな開発も可能です

Express.js

Express.js2

node.jsを用いるフレームワーク。本来フロントエンドで使用するjavascriptですが、こちらはサーバーサイドで使用します。そのフレームワークです。2009年に生まれたnode.jsはリアルタイム通信に強い分野です。高速トラフィックの処理等の分野でも有効ですね。主なサービスはPAYPALなどが有名ですね。軽量で高速開発をやりやすく、導入も割と楽な部分もあります。ただこちらのフレームワークの場合、一つだけ使いづらい点があり、若干どこを編集したら使いやすいかわかりづらい点がございます。またnode.js自体がかなり癖のあるプログラミング言語である為、習得にやや困難な部分もございます
