Vueインストール

Vue

sudo apt-get update

sudo apt-get upgrade

Vueをインストールします。

sudo apt-get install npm

sudo npm install -g vue-cli

Vueプロジェクトを導入します。

vue init webpack mysite

　

Vueプロジェクト

Vueプロジェクトのディレクトリへ移動します

cd mysite

Vueプロジェクトにインストールコマンド

npm install

Vueプロジェクトを起動します

npm run dev

http://localhost:8080/#/

左図のような画面がでれば成功です!!

Nuxtインストール

Nuxt

今回は結構早かったですが、次のNuxt.jsも記載いたします

Nuxt

Vue.js(CLI)がインストールされていないといけませんので、既にインストールしたあととして記述して参ります。

Vueプロジェクトを導入します。

vue init nuxt-community/starter-template lovejs

プロジェクトのディレクトリへ移動します

cd lovejs

Nuxt

プロジェクトの中に再度インストールコマンド

npm install

プロジェクトを起動します

npm run dev

画面が変化していきます。

port番号がよく変わらるのでご注意ください
