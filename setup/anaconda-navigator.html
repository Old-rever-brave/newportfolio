Anacondaインストール

Anaconda

Anacondaをダウンロードします。

https://www.anaconda.com/distribution/#download-section

2.7か3.7のバージョンがあります。のでお好きな方で。3.7の手順のほうで記載していきます

bashで解凍しますが、ホームディレクトリにコピペか切り取りでもっていった方が多分楽です。

bash Anaconda3-2019.03-Linux-x86_64.sh

所有権変更をします

chmod 755 anaconda3/bin/conda

MySQL導入手順

PATHを通します。

which python

which ipython

which pip

vi ~/.bashrc

↑↑↑※要注意

エディタの画面になるのでインサート（挿入）をするコマンドiを押してください。そして以下をコピペ

export PATH=~/anaconda3/bin:$PATH

インストール

②下記の手順で強制的に保存します。

esc

:wq!

condaコマンドにて一度、古いデータを綺麗にします。

conda config –remove channels conda-forge

anacondaを開きます。

anaconda-navigator

図のような画面がでてきます。
