djangoインストール

　　　

pipiコマンドでdjangoをインストールします

pip install django

　

Djangoのプロジェクトを開発します。

django-admin startproject django_web

プロジェクトのディレクトリへ移動します。

cd django_web

ランサーバーを起動します

python manage.py runserver

ターミナルに以下のリンクが表示されます。

起動まで

http://127.0.0.1:8000

上記のリンクを開きます

するとこんな画像がでてきます。
