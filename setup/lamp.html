lampインストール

php

PHPに関連するパッケージをインストール、設定することでphp7がインストールされます

sudo apt-get install apache2 php php-gd mysql-server php-mysql

sudo apt-get install libapache2-mod-php

Apache（webサーバー）をインストール。

sudo apt-get install apache2

下記自動起動の設定します

Apache

sudo systemctl enable apache2

Apacheを再起動します。

sudo systemctl restart apache2

ドキュメントページにいきます。

cd /var/www/html

こちらの場所でファイルを作ります。

sudo touch index.html

御自分のIPアドレスを直接入力、後こちらの画面がでれば成功です

MySQL導入手順

MySQLをインストールします

sudo apt-get install mysql-server

MySQLを設定をします

sudo mysql_secure_installation

新しいパスワードを設定します。

MySQL導入手順②

一旦、ctrl+zでターミナルを閉じて、もう一度いかのコマンド。

mysql_secure_installation

先程のパスワードを二回入力します。

そして、すべての質問にYで答えましょう。

ログインコマンドを入力

sudo mysql -u root -p

先程のパスワードを入力すると、図の画面が現れ成功です
