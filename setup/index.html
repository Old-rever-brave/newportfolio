以下は全く触ったことのない方々向けに記載しております。

アップデートをかけます

sudo apt-get update

アップグレードをかけます

sudo apt-get upgrade

ファイヤーウォールを切ります

sudo ufw disable

SELINUXを切ります。

sudo service apparmor stop

sudo apt install net-tools

gitのインストール

sudo apt install git

sshのインストール

sudo apt install openssh-server

sudo systemctl start ssh

だいたいこれで、一通りは不自由ないかと思われます。

注意点：ubuntuでアップグレードをかける際、ubuntuソフトウェアを開かないようお願いいたします。※最悪の場合、バグがでます。
